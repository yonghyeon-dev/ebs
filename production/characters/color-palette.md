# 마이크로 보이저 — 캐릭터 및 레이어별 색상 팔레트

> 본 문서는 마이크로 보이저 전 에피소드의 캐릭터 색상 및 에피소드별 색감 톤을 정의한다.
> 모든 HEX 코드는 ComfyUI/Photoshop/After Effects에서 직접 사용 가능하다.
> 작품소개서 p.8 비주얼 디렉션 기준.

---

## 1. 캐릭터별 색상 팔레트

### 1.1 박초은 (초은이)

7세 소녀. 밝고 활기찬 색감. 현실/미시 양쪽 등장.

#### 현실 세계 팔레트 (수채화 톤)

| 부위 | 색상명 | HEX | Danbooru 태그 | 용도 |
|------|--------|-----|-------------|------|
| 머리카락 | 소프트 블랙 | `#2C2C2C` | `black hair, bob cut` | 완전한 검정이 아닌 따뜻한 톤의 검정. 수채화 하이라이트로 `#3D4F6B` 블루 계열 반사 |
| 땋은머리 구슬 | 스카이 블루 | `#87CEEB` | `blue bead hair tie, single side braid` | 오른쪽 작은 땋은머리에 묶은 하늘색 구슬. 캐릭터 식별 포인트 |
| 탐험 조끼 | 파스텔 민트 | `#98FB98` | `green vest, light green` | 현장 탐험가 느낌의 민트색 조끼. 캐릭터 아이덴티티 핵심 |
| 셔츠 | 크림 화이트 | `#FFFEF2` | `cream shirt, long sleeves` | 조끼 안에 입는 따뜻한 크림빛 긴팔 셔츠 |
| 카키 반바지 | 카키 | `#C3B091` | `khaki shorts, cargo shorts` | 큰 주머니 달린 카고 반바지. 자연탐험가 이미지 |
| 양말 (줄무늬) | 스트라이프 블루 | `#4D96FF` | `striped socks, blue socks` | 짝짝이 양말 중 줄무늬 파랑 (7세다운 엉뚱함) |
| 양말 (점무늬) | 도트 그린 | `#6BCB77` | `polka dot socks, green socks` | 짝짝이 양말 중 점무늬 초록 |
| 부츠 | 웜 브라운 | `#8B6914` | `brown boots, lace-up boots` | 낡고 신어서 편해진 갈색 끈 부츠. 밑바닥에 흙 묻음 |
| 피부 | 피치 퍼프 | `#FFDAB9` | (자동) | 따뜻한 한국인 아동 피부톤 (수채화) |
| 볼 | 로즈 핑크 | `#FFB6C1` | `blush` | 수채화 특유의 볼 터치. 건강한 혈색 |
| 돋보기 테 | 앤틱 골드 | `#B8860B` | `magnifying glass, brass, leather cord` | 황동 프레임. 세월의 녹청. 가죽 끈으로 목에 걸림 |
| 돋보기 렌즈 | 투명 + 프리즘 | `#E8F4FD` (반사) | `glass reflection, rainbow prism` | 투명 유리 + 빛 굴절 시 무지개 프리즘 |

#### 미시 세계 팔레트 (프리즘 반사 추가)

현실 세계 팔레트를 기본으로 유지하되, 다음 변형을 적용한다.

| 변형 요소 | 적용 방식 |
|-----------|----------|
| 조끼 표면 | `#98FB98` 유지 + 프리즘 환경 반사 오버레이 (opacity 20~30%), 무지개빛 빛 입자 반사 |
| 피부 | 기본 피치 유지 + 서브서피스 스캐터링 (내부에서 은은한 `#FFE4E1` 발광) |
| 구슬 땋은머리 | `#87CEEB` → 프리즘 환경에서 무지개빛 반사로 다채로워짐 |
| 눈 하이라이트 | 흰색 2점 → 프리즘 무지개색 2점 (`#FF6B6B` + `#4D96FF` 등 환경색 반영) |
| 돋보기 | 황동 테 유지 + 포착된 빛으로 은은하게 발광 (`#FFD700` 글로우) |
| 반바지/부츠 | 기본색 유지 + 가장자리 크로매틱 에버레이션 힌트 |

### 1.2 할아버지 (박 박사)

은퇴한 과학자. 따뜻하고 인자한 색감. 현실 세계 전용.

#### 현실 세계 팔레트 (수채화 톤)

| 부위 | 색상명 | HEX | Danbooru 태그 | 용도 |
|------|--------|-----|-------------|------|
| 머리카락 | 실버 화이트 | `#C0C0C0` | `silver hair, white hair, fluffy side hair` | 은백색. 윗머리 빈약, 옆머리 민들레 홀씨 같은 솜털. 수채화 `#D8D8D8` 하이라이트 |
| 안경테 | 와이어 실버 | `#A9A9A9` | `round glasses, wire-rimmed glasses` | 둥근 와이어 프레임. 넓은 코 위에 낮게 걸침 |
| 안경 렌즈 | 투명 + 글레어 | `#E8F4FD` (반사) | `glasses on nose` | 기본 투명. 빛 반사 시 연한 하늘빛 |
| 셔츠 | 빛바랜 체크 | `#D2C4B0` (베이스) | `plaid shirt, flannel shirt, red plaid` | 빛바랜 레드-크림 체크 플란넬 셔츠 |
| 카디건 | 카멜 브라운 | `#C19A6B` | `brown cardigan, corduroy` | 카멜 브라운 코르듀로이. 팔꿈치에 갈색 가죽 패치 |
| 팔꿈치 패치 | 다크 브라운 | `#654321` | `elbow patches, leather patches` | 가죽 소재. 학자/발명가 느낌 강조 |
| 바지 | 다크 브라운 | `#5C4033` | `dark brown pants` | 진한 갈색 울 바지. 편안한 핏 |
| 실내화 | 브라운 레더 | `#8B6914` | `brown slippers` | 갈색 가죽 실내화 |
| 피부 | 밀색 베이지 | `#F5DEB3` | (자동) | 따뜻한 노인 피부톤 (수채화). 손등에 혈관 표현 |
| 잔주름 | 라이트 브라운 | `#D2B48C` | `smile lines, crow's feet, wrinkles` | 깊은 미소 주름, 눈가 잔주름(까마귀 발). 수채화 라인 |
| 눈썹 | 실버 와일드 | `#B0B0B0` | `bushy eyebrows, silver eyebrows` | 덥수룩하고 약간 헝클어진 은색 눈썹 |

#### 미시 세계 등장 시 색감 (간접 등장)

할아버지는 미시 세계에 직접 등장하지 않으나, 다음 상황에서 색감이 반영된다.

| 등장 방식 | 적용 색감 |
|-----------|----------|
| 무전기 음파 시각화 | 빛의 물결: `#F5DEB3`(피부) → `#FFD700`(따뜻한 골드) 그라데이션 파동. 동심원형 황금빛 빛 물결 |
| 물방울 표면 반사 | 왜곡된 얼굴: `#C0C0C0`(은발) + `#A9A9A9`(와이어 안경) 힌트가 물 표면 구면 굴절과 혼합 |
| 가이드 빛 | 따뜻한 조명: `#FFF8DC`(콘실크) — 할아버지의 도움을 상징하는 따뜻한 앰버+골드 빛 |

### 1.3 뭉이 (물곰/타디그레이드)

미시 세계 전용 캐릭터. 투명하고 발광하는 색감.

#### 미시 세계 팔레트

| 부위 | 색상명 | HEX | Danbooru 태그 | 용도 |
|------|--------|-----|-------------|------|
| 몸체 주요 | 아이스 블루 | `#D4EEFF` | `translucent, semi-transparent, pale blue-green body` | 반투명 (60~70% opacity). 물곰의 투명 특성 |
| 몸체 그림자 | 소프트 스카이 | `#A8D8EA` | (자동) | 투명체 아래쪽/접힌 부분 그림자 |
| 내부 장기 힌트 | 파스텔 민트 | `#A8E6CF` | `internal glow, bioluminescent` | 내부가 은은하게 비치는 효과 |
| 눈 (동공) | 딥 블랙 | `#2B2B2B` | `large eyes, round eyes, dark eyes` | 크고 둥근 눈. 캐릭터성 핵심 |
| 눈 (하이라이트) | 퓨어 화이트 | `#FFFFFF` | `sparkle in eyes` | 큰 하이라이트 1점 + 작은 하이라이트 1점 |
| 볼/포인트 | 체리블로섬 | `#FFB7C5` | (자동) | 양볼에 은은한 핑크. 귀여움 강조 |
| 다리 | 트랜스 블루 | `#C8E6FF` | `8 legs, stubby legs, rounded claws` | 반투명 (70% opacity). 몸체보다 약간 진한 블루 |
| 바이오 글로우 | 민트 글로우 | `#7FFFD4` | `bioluminescent, glowing` | 몸체 가장자리 발광 효과 (외곽선 대체) |

#### 에피소드별 뭉이 색상 변형

뭉이는 에피소드별 환경에 따라 바이오 글로우 색상이 미세하게 변한다 (기본 체색은 고정).

| 에피소드 | 바이오 글로우 변형 | HEX |
|----------|------------------|-----|
| 1화 물방울 | 민트 블루 (기본) | `#7FFFD4` |
| 2화 꽃잎 | 라벤더 핑크 | `#DDA0DD` |
| 3화 나비 날개 | 이리데슨트 그린 | `#98FB98` |
| 4화 흙 속 | 웜 앰버 | `#FFBF00` |
| 5화 눈 결정 | 아이스 화이트 | `#F0FFFF` |
| 6화 산호 왕국 | 네온 시안 | `#00FFFF` |
| 7화 과일 세포 | 라임 글로우 | `#ADFF2F` |
| 8화 나뭇잎 | 에메랄드 | `#50C878` |
| 9화 거미줄 | 실버 글로우 | `#E8E8E8` |
| 10화 빗방울 | 스카이 블루 | `#87CEEB` |

## 2. 레이어별 전체 색상 가이드

### 2.1 현실 세계 팔레트 (웜톤 기반)

모든 에피소드의 현실 세계 씬에 공통 적용되는 환경 색상.

| 요소 | 색상명 | HEX | 비고 |
|------|--------|-----|------|
| 하늘 | 소프트 스카이 | `#87CEEB` | 맑은 날 기본. 구름은 `#F5F5F5` |
| 햇빛 | 골든 아워 | `#FFD700` → `#FFF8DC` | 그라데이션. 광선은 `#FFFACD` |
| 나무/잎 | 소프트 그린 | `#8FBC8F` | 수채화 녹색. 짙은 부분 `#6B8E23` |
| 꽃 | 파스텔 핑크 | `#FFB7C5` | 기본 꽃색. 에피소드별 변형 가능 |
| 흙/땅 | 웜 어스 | `#D2B48C` | 따뜻한 황토색 |
| 물 | 크리스탈 블루 | `#B0E0E6` | 투명하고 반짝이는 물 표현 |
| 실내 (할아버지 연구실) | 웜 크림 | `#FFF8DC` | 따뜻한 조명의 아늑한 공간 |

### 2.2 미시 세계 팔레트 (쿨톤 + 프리즘 기반)

모든 에피소드의 미시 세계 씬에 공통 적용되는 기본 환경 색상. 에피소드별 주도색은 이 위에 오버레이된다.

| 요소 | 색상명 | HEX | 비고 |
|------|--------|-----|------|
| 기본 환경광 | 트랜스 블루 | `#E0F0FF` | 모든 미시 세계의 베이스 톤 |
| 프리즘 스펙트럼 빨강 | 스펙트럼 레드 | `#FF4444` | 빛 굴절 스펙트럼 |
| 프리즘 스펙트럼 주황 | 스펙트럼 오렌지 | `#FF8C00` | 빛 굴절 스펙트럼 |
| 프리즘 스펙트럼 노랑 | 스펙트럼 옐로우 | `#FFD700` | 빛 굴절 스펙트럼 |
| 프리즘 스펙트럼 초록 | 스펙트럼 그린 | `#32CD32` | 빛 굴절 스펙트럼 |
| 프리즘 스펙트럼 파랑 | 스펙트럼 블루 | `#4169E1` | 빛 굴절 스펙트럼 |
| 프리즘 스펙트럼 남색 | 스펙트럼 인디고 | `#4B0082` | 빛 굴절 스펙트럼 |
| 프리즘 스펙트럼 보라 | 스펙트럼 바이올렛 | `#8B00FF` | 빛 굴절 스펙트럼 |
| 발광 효과 | 바이오 글로우 | `#7FFFD4` | 생물체 자체 발광 기본색 |
| 분자 구조 (산소) | 옥시젠 블루 | `#4682B4` | H2O 분자 시각화: 큰 구(산소) |
| 분자 구조 (수소) | 하이드로 화이트 | `#F0F8FF` | H2O 분자 시각화: 작은 구 2개(수소) |
| 분자 결합선 | 본드 라이트 | `#ADD8E6` | 수소결합 시각화 연결선 |

## 3. 에피소드별 색감 톤 가이드 (1~10화)

작품소개서 p.8 비주얼 디렉션 기준. 각 에피소드의 미시 세계 고유 색감을 정의한다.

### EP01 — 물방울 속 우주

| 항목 | 값 |
|------|-----|
| **비주얼 키워드** | 프리즘 굴절, 투명한 구체, 빛의 스펙트럼 |
| **색감 톤** | 무지개 + 투명 블루 |
| **주도색** | `#87CEEB` 투명 블루 |
| **보조색** | `#E0F0FF` 라이트 아이스, `#B0E0E6` 파우더 블루 |
| **강조색** | 무지개 스펙트럼 전체 (`#FF4444` ~ `#8B00FF`) |
| **금지 색감** | 어두운 톤, 불투명한 색, 어스톤 |
| **ComfyUI 색감 키워드** | `transparent blue base, rainbow prismatic overlay, crystal clear, light spectrum bands, iridescent highlights` |

### EP02 — 꽃잎 위의 별빛

| 항목 | 값 |
|------|-----|
| **비주얼 키워드** | 꽃잎 미세구조, 꽃가루 입자, 벨벳 텍스처 |
| **색감 톤** | 핑크-퍼플 그라데이션 |
| **주도색** | `#FF69B4` 핫 핑크, `#9B59B6` 아메시스트 퍼플 |
| **보조색** | `#DDA0DD` 플럼, `#FFB6C1` 라이트 핑크 |
| **강조색** | `#FFD700` 꽃가루 골드, `#FFFFFF` 별빛 하이라이트 |
| **금지 색감** | 차가운 블루 계열, 어스톤, 네온 그린 |
| **ComfyUI 색감 키워드** | `pink-purple gradient, velvet petal texture, golden pollen particles, starlight sparkle, warm magenta, soft floral tones` |

### EP03 — 나비 날개의 비밀

| 항목 | 값 |
|------|-----|
| **비주얼 키워드** | 나노 구조색, 비늘 패턴, 회절 무늬 |
| **색감 톤** | 이리데슨트 (무지개빛 반사) |
| **주도색** | `#E0B0FF` 모브, `#98FB98` 페일 그린 |
| **보조색** | `#DDA0DD` 플럼, `#87CEEB` 스카이 블루 |
| **강조색** | 홀로그래픽 시프트 — 보는 각도에 따라 `#FF69B4` -> `#4169E1` -> `#32CD32` 변화 |
| **금지 색감** | 단색 고정 (반드시 색상 변화가 있어야 함), 탁한 색 |
| **ComfyUI 색감 키워드** | `iridescent surface, holographic color shift, structural coloration, diffraction grating rainbow, nano-scale interference pattern, oil-on-water color effect` |

### EP04 — 흙 속 지하도시

| 항목 | 값 |
|------|-----|
| **비주얼 키워드** | 토양 입자, 균사 네트워크, 지하 동굴 |

## 4. 색상 전환 규칙

### 4.1 현실 -> 미시 색감 전환

```
현실 세계 (웜톤)
  골든 옐로우 #FFD700
  피치 #FFDAB9
  소프트 그린 #8FBC8F
       |
       | 전환 (빛 입자 분해)
       | 웜 → 뉴트럴 → 쿨
       v
미시 세계 (에피소드별 톤)
  트랜스 블루 #E0F0FF (기본)
  + 에피소드별 주도색
  + 프리즘 스펙트럼 오버레이
```

### 4.2 에피소드 내 색감 일관성 규칙

- 미시 세계 진입 후 첫 5초: 기본 트랜스 블루(`#E0F0FF`)에서 에피소드 주도색으로 점진 전환
- 에피소드 중반: 주도색 100% 적용. 강조색은 과학 포인트/감정 고조 시 사용
- 복귀 직전: 주도색이 서서히 트랜스 블루로 돌아감 → 현실 세계 웜톤으로 전환

### 4.3 색감 채도/명도 가이드

| 상황 | 채도 | 명도 | 예시 |
|------|------|------|------|
| 평화로운 탐험 | 중간 (50~65%) | 높음 (70~85%) | 기본 씬 |
| 과학 발견 순간 | 높음 (70~85%) | 높음 (80~95%) | 프리즘 효과 강화, 하이라이트 증가 |
| 위기/긴장 | 낮음 (30~45%) | 낮음 (40~55%) | 색이 탁해지고 어두워짐 |
| 해결/감동 | 높음 (75~90%) | 매우 높음 (85~100%) | 빛이 폭발하듯 밝아짐 |
| 현실 세계 (전체) | 낮음~중간 (35~55%) | 높음 (75~90%) | 수채화 파스텔 톤 |

## 5. After Effects / DaVinci Resolve 색보정 가이드

생성된 이미지/영상을 합성할 때 참조하는 색보정 기준값.

### 5.1 현실 세계 색보정

| 파라미터 | 값 | 비고 |
|----------|-----|------|
| Color Temperature | +200~+500K (따뜻하게) | 골든아워 느낌 |
| Tint | +5~+10 (약간 마젠타) | 수채화 따뜻함 |
| Saturation | -10~-15% | 과포화 방지, 파스텔 톤 유지 |

| Grain | 추가 (미세) | 종이 질감 시뮬레이션 |

## 6. 색상 접근성 메모

아동 대상 콘텐츠이므로 다음 사항을 고려한다.

- **색각이상 대응**: 정보 전달 시 색상만으로 구분하지 않음 (형태/패턴 병행)
- **깜박임 제한**: 초당 3회 이상 급격한 색상 변화 금지 (광과민성 발작 방지, Ofcom 가이드라인)
- **대비 유지**: 캐릭터와 배경 간 충분한 명도 대비 유지 (캐릭터 식별성)
- **어두운 씬 제한**: EP06(산호 왕국의 밤)도 완전한 암흑은 피하고 네온 발광으로 최소 밝기 보장
