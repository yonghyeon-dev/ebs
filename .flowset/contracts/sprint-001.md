## 수용 기준
- [ ] 각 캐릭터 프롬프트에 **외형 묘사**(체형, 얼굴, 의상, 소품)가 구체적으로 포함
- [ ] 초은이: 정면/측면/후면 + 표정 3종(호기심/놀람/기쁨) + 돋보기 소품 포함
- [ ] 할아버지: 정면/측면 + 표정 2종(미소/놀람) + 무전기·마이크로렌즈 소품 포함
- [ ] 뭉이: 정면/측면/위에서 + 동작 3종(걷기/서핑/인사) + 8개 다리·투명 몸체 포함
- [ ] 모든 프롬프트에 **네거티브 프롬프트** 포함
- [ ] 모든 프롬프트에 **권장 파라미터** 포함 (모델, 샘플러, CFG, 해상도, 시드 가이드)
- [ ] 현실 세계(수채화풍)와 미시 세계(초현실) 두 스타일 버전 구분
- [ ] 프롬프트가 영어로 작성되고, 한글 주석으로 의도 설명
